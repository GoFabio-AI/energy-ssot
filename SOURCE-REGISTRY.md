# Source Registry

Registro delle fonti approvate per il SSOT energia.

## Regole di ammissione

### Ammesse
- ARERA
- Normattiva
- Gazzetta Ufficiale
- Acquirente Unico / SII
- Terna
- Agenzia delle Entrate
- ADM
- INPS
- Altri enti ufficiali rilevanti per obblighi normativi o fiscali

### Ammesse solo come supporto secondario
- FAQ ufficiali
- manuali operativi ufficiali
- documentazione tecnica ufficiale collegata a una fonte primaria

### Non ammesse nel SSOT
- blog
- forum
- materiale marketing
- riassunti AI
- fonti non ufficiali

## Template voce fonte
- ID fonte
- Ente
- Titolo
- URL
- Tipo fonte
- Stato: vigente / da verificare / superata
- Ambito
- Data pubblicazione
- Data ultima verifica
- Note
